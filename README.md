# つながり日記（プロダクト名）


## 製品概要
音声と位置情報から子供のその日の活動記録を絵日記として残すことができるシステム

## テーマ
つながり ✖ Tech

### 背景(製品開発のきっかけ、課題等）
共働き家庭の増加による親子間のコミュニケーション時間の減少

### ターゲット
メインターゲット：3歳〜10歳のお子さんを持つ共働き世帯の親（特に子育て中のデジタルネイティブ世代）

### 製品説明（具体的な製品の説明）
上記の課題を解決するために，親が子供の活動を子供風の絵日記で確認し，具体的なコミュニケーションの提供をメインとする．
また，その副産物として，日々の何気ない活動を絵日記に残すことで，運動会などのイベントではない小さな日常の思い出を残し，見返すことができるシステム．
子供の音声と位置情報をデータベースに格納し，位置情報と会話の内容から，どこで何をしているか(イベント)を生成AIを用いて推測する．
複数の推定したイベントから，生成AIが重要な項目をピックアップし，絵日記（画像と文章）を生成する．
親はカレンダーの日付をタップすることで絵日記を見ることができる．

### 特徴
#### 特徴1
家族の温かさが感じられるような，優しいUI
#### 特徴2
子供が自身で作成した絵日記を実際に読んでいるかのような体験の提供．子供がクレヨンで描く絵の感じと，子供が書く字を再現
#### 特徴3
過去に生成した絵日記がカレンダー上で見ることができる
#### 特徴4
生成AIに対するプロンプトの工夫したことにより，生成内容や文字数などの条件が大きく変わらないように工夫
#### 特徴5
処理時間の速さ，データベースの行数にもよるが，生成AIを3回使用で平均処理時間30秒，長くても1分程度の速さ

### 解決出来ること
具体的なコミュニケーションの提供
大きなイベントではない日常的な子供の成長の記録
家族だけでなく親戚にも共有することで，遠く離れていても身近に感じることが可能

### 今後の展望
対象人物の拡大（例えば認知症の方やご老人など）や，その人物の行動把握
親がリアルタイムに子供の位置を把握できるような見守り機能の実装

### 注力したこと（こだわり等）
* 生成AIを複数回使用しても，同じような条件の出力（文章の体裁や文字数）を再現するためのプロンプト
* 子供が書いたような絵日記の生成
* 位置情報（緯度経度）から周辺にある施設（「〇〇公園」「△△保育園」）への変換
